## 완료된 작업

### 1단계: Auth ✅
- 이메일/구글/카카오 로그인
- JWT (access + refresh) 토큰 관리
- 회원가입, 로그아웃, 비밀번호 찾기, 토큰 갱신

### 2단계: 트래킹 (GPS, 경로, 거리, 쓰레기 수거) ✅

**모델**: TrackingSession, TrackingPoint, TrashPoint, Place

**엔드포인트 14개**
```
POST   /api/tracking/sessions
GET    /api/tracking/sessions/active
POST   /api/tracking/sessions/{id}/points
POST   /api/tracking/sessions/{id}/end
GET    /api/tracking/sessions
GET    /api/tracking/sessions/{id}
PATCH  /api/tracking/sessions/{id}
DELETE /api/tracking/sessions/{id}
POST   /api/tracking/sessions/{id}/pause
POST   /api/tracking/sessions/{id}/resume
POST   /api/tracking/sessions/{id}/trash-points
GET    /api/tracking/sessions/{id}/trash-points
GET    /api/users/me/stats
GET    /api/places/search
```

**핵심 기능**
- Haversine 거리 계산 (GPS 포인트 누적 → 종료 시 재계산 보정)
- 일시정지/재개: `pause_duration_seconds` 누적, `duration = 전체경과 - pause_duration`
- GPS 포인트 5초 간격 저장
- 1시간 초과 시 active/paused 세션 자동 expired 전환
- 실시간 쓰레기 마킹 (지도 마커용)
- 네이버 지역 검색 프록시 (X-Naver-Client-Id/Secret)
- 사용자 누적 통계 (completed 세션 합산, level → 대표값 변환)

### 3단계: 피드/포스트 ✅

**모델**: Post, Comment, Like

**엔드포인트 13개**
```
POST   /api/images/upload
GET    /uploads/{filename}

POST   /api/posts
GET    /api/posts
GET    /api/posts/{id}
PATCH  /api/posts/{id}
DELETE /api/posts/{id}

POST   /api/posts/{id}/like
DELETE /api/posts/{id}/like

GET    /api/posts/{id}/comments
POST   /api/posts/{id}/comments
PATCH  /api/comments/{id}
DELETE /api/comments/{id}
```

**포스트 타입**
| 타입 | images | tracking_id | is_verified |
|------|--------|-------------|-------------|
| A    | ✅      | ✅           | true (인증 배지) |
| B    | ❌      | ✅           | true (인증 배지) |
| C    | ✅      | ❌           | false |

**핵심 기능**
- 이미지 로컬 저장 (`uploads/` 디렉터리) + StaticFiles 서빙
- 지원 포맷: jpg, jpeg, png, heic, heif, webp
- 포스트 생성 시 `tracking_id`가 있으면 TrackingSession.post_id 역참조 자동 설정
- 좋아요/댓글 수 카운터 동기화 (like_count, comment_count)
- images OR tracking_id 중 하나 이상 필수 (둘 다 없으면 422)
