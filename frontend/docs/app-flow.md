# MetaPlogging 앱 흐름 정의

## 피드 포스트 구조

### 포스트 타입

포스트는 **글 필수**, 이외 아래 3가지 형태로 구성된다.

| 타입 | 구성 | 인증 배지 | 포인트 |
|---|---|---|---|
| A | 글 + 사진 + 트래킹 | ✅ 🌿 인증 | 높음 |
| B | 글 + 트래킹 | ✅ 🌿 인증 | 높음 |
| C | 글 + 사진 | ❌ | 낮음 |

> 글만 올리는 것은 불가. 사진 또는 트래킹 기록 중 하나는 반드시 포함.

---

## 포스트 업로드 흐름

```
업로드 시작
│
├─ 글 입력 (필수)
│
├─ 사진 첨부 (선택)
│   └─ 갤러리에서 선택 or 카메라 촬영
│
├─ 트래킹 기록 연결 (선택)
│   ├─ 오늘 트래킹 기록이 있으면 → "이번 활동과 연결할까요?" 자동 제안
│   └─ 없으면 → 기록 없이 업로드 가능
│
└─ 업로드
    ├─ 사진 + 트래킹 있으면 → 타입 A (🌿 인증)
    ├─ 트래킹만 있으면     → 타입 B (🌿 인증)
    └─ 사진만 있으면       → 타입 C (일반)
```

## 피드 표시

### 카드 레이아웃 (타입별)

```
[타입 A: 사진 + 트래킹]
┌─────────────────────────┐
│ 🌿 인증  작성자 · 시간   │
├─────────────────────────┤
│ 📷 이미지 (carousel)    │
├─────────────────────────┤
│ 📊 3.2km | 24개 | 42분 │
├─────────────────────────┤
│ 글 내용                 │
│ #태그                   │
└─────────────────────────┘

[타입 B: 트래킹만]
┌─────────────────────────┐
│ 🌿 인증  작성자 · 시간   │
├─────────────────────────┤
│ 📊 3.2km | 24개 | 42분 │
├─────────────────────────┤
│ 글 내용                 │
│ #태그                   │
└─────────────────────────┘

[타입 C: 사진만]
┌─────────────────────────┐
│ 작성자 · 시간           │
├─────────────────────────┤
│ 📷 이미지 (carousel)    │
├─────────────────────────┤
│ 글 내용                 │
│ #태그                   │
└─────────────────────────┘
```

### 피드 정렬 기준

1. **기본 정렬**: 트래킹 인증 포스트(A, B) 우선 → 이후 최신순
2. **필터 옵션**: "활동 인증만 보기" 토글

## 트래킹 & 게이미피케이션

### 트래킹 기록이 포함된 포스트의 혜택

- 🌿 인증 배지 표시 (신뢰성)
- 포인트 2배 지급
- 주간/월간 랭킹 포함
- 프로필 통계(총 거리, 수거량, 활동 횟수)에 카운트

### 트래킹 기록이 없는 포스트

- 배지 없음
- 기본 포인트
- 랭킹 제외
- 프로필 통계 미반영

## 트래킹 세션 흐름

```
플로깅 시작
│
├─ GPS 트래킹 시작
│   ├─ 실시간 거리 측정
│   ├─ 경로 지도 표시
│   └─ 쓰레기 수거량 수동 기록
│
└─ 플로깅 종료
    ├─ 트래킹 결과 요약 화면
    └─ "피드에 공유할까요?" 제안
        ├─ 예 → 글/사진 추가 후 업로드 (타입 A or B)
        └─ 아니오 → 개인 기록으로만 저장
```

## 페이지별 역할 정의

| 페이지 | 역할 |
|---|---|
| 홈 | 커뮤니티 현황, 나의 통계 요약, 바로가기 |
| 피드 | 모든 포스트 (타입 A/B/C 혼합), 활동 공유 |
| 플로깅 | 트래킹 시작/종료, 개인 활동 기록 |
| 프로필 | 게이미피케이션 (레벨, 배지), 나의 포스트 |
